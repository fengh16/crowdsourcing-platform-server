    module:models/jwt~ Jwt

    JWT model，还包含以下额外的属性：

    • JsonWebTokenError: 所有JWT错误的基类，可能会在verify()的时候发生（比如签名错误）
    • NotBeforeError: 本项目不会用到
    • TokenExpiredError: JWT过期，可能会在verify()的时候发生
    • RevokedError: JWT被撤回，可能会在verify()的时候发生

    该类实例化的对象为global.jwt
